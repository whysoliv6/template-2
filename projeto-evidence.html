Evidence

Marketplace white label modular para venda de suplementos

Visão Geral

ClienteFarmácia de Manipulação Evidência
SetorFarmácia de manipulação, suplementos
EntregaSistema de marketplace white label, modular, com site incluso
EscopoWeb e Desktop

Desafio

A Farmácia Evidência queria criar um modelo white label completo de marketplace, para que médicos, profissionais de saúde e influenciadores pudessem vender suas próprias linhas de suplementos (creatina, whey, pré-treino), com a farmácia cuidando de toda a produção e fornecendo o site e o sistema de marketplace.

Processo

Esqueleto

Sistema modular pensado para que o próprio usuário revendedor trocasse foto, banner, domínio e cores sem depender de desenvolvimento novo a cada cliente.

Superfície

Construção baseada em tokenização: mudar um parâmetro de cor já reflete em toda a interface, com modelos de banner e copy intercambiáveis entre revendedores.

Prova de Fogo

Fluxo de compra desenhado para funcionar mesmo sem login, priorizando simplicidade no checkout em vez de depender de autenticação para puxar dados do usuário.

Decisões estratégicas

  • Design orientado a tokens. Toda a base visual do marketplace foi construída para ser trocada por parâmetro, não por retrabalho manual, o que viabiliza escalar para vários revendedores com o mesmo sistema.
  • Checkout sem dependência de login. Mesmo sendo possível fazer login, o fluxo de compra foi desenhado para funcionar de forma simples sem essa exigência, reduzindo fricção na conversão.

Resultado

Sistema de marketplace white label entregue em versão web e desktop, com arquitetura modular pronta para replicação entre diferentes revendedores.
